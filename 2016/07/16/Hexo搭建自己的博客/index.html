  • 目标：想要在浏览器在hexo server在正常启动后显示出预览界面。问题：启动hexo server 提示信息显示 hexo is running, 不能显示页面。问题排查1,使用端口被占用，改变端口4000 –> 5000 –> 8000 –> 4001 –> 5555等等都未能成功。问题排查2：wget 本地网址 提示没有到主机的路由。于是尝试关闭防火墙。没用。

  • 问题排查思路

描述：
hexo is running on http://localhost:4000 但是浏览器提示 无法连接。

那么，
1、在这是一个怎样的过程？2、这个过程的那个环节出了问题？

试答：
这是一个网络通信过程，浏览器像服务器发送request
